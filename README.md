# Project-2---BIOSTAT-721

This github repository contains three csv files (AQ_2018, AQ_2019, and AQ_2020)
that contain data on the air pollutants CO, O3, and PM2.5 in Durham County, 
North Carolina from the EPA.

The Project2.Rmd file is an R markdown file which contains a function to clean 
this data as well as code to plot the air pollutant data. The Project2.docx 
contains a Word document presenting the plots created in the markdown file as 
well as a link to the github repository.
